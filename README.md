# 📚 AttendMe

AttendMe is a full-stack Attendance Management System designed for schools to digitize and streamline classroom attendance tracking. It provides secure authentication, role-based access control, real-time updates, and cross-platform access through web and mobile applications.

Built using modern technologies including Spring Boot, ReactJS, and Android Kotlin, AttendMe ensures secure, scalable, and efficient attendance management for administrators and teachers.

🎯 Project Overview

Traditional attendance systems rely on manual records, which can be time-consuming and prone to errors. AttendMe modernizes this process by offering:

Digital attendance recording
Secure user authentication
Centralized data management
Real-time attendance monitoring
Web and mobile platform support

👥 User Roles
👨‍💼 Admin

Manage teachers and students
Manage classes and subjects
View attendance reports
Monitor attendance trends

👩‍🏫 Teacher

View assigned classes
Create attendance sessions
Mark student attendance (Present/Absent)
View attendance history
